# 1. Subsidiary Discovery

Before enumerating the primary domain, map all subsidiaries — they may be easier targets and are often in scope.

| Source | Method |
|--------|--------|
| Google | Manual search: `"<company>" subsidiaries`, `site:crunchbase.com "<company>"` |
| Crunchbase | https://www.crunchbase.com — search company, check "subsidiaries" tab |
| Aleph (OCCRP) | https://aleph.occrp.org — leaked docs, corporate registries, sanctions data |
| Own knowledge base | Bug bounty program scope, prior reports, LinkedIn org chart |
